Graph.edu

This is a diploma project of Viktor Maksimenko, ONPU student.
This web app displays the graph of computer science subjects. Edges between graph nodes are dependencies between subjects. Click node to highlight dependencies of single discipline. For more info about subject double click it's node.
The graph is scalable and rotateble. Use range elements to rotate it from 0 to 360 degres and scale from 0.2 to 10 size. You can also scale grapf with mouse scroll.
You may also use choose one of layouts or drag nodes to positions you prefer.
Use config flags for creating the best view and save it as SVG, JPG, PNG, GIF or TIFF.
For graph drawing we use Sigma.js - a JavaScript library dedicated to graph drawing.
For more details view our project on github
